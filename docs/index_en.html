Description

The ZXTRES, ZXTRES+ and ZXTRES++ are a continuation of the ZX-Uno hardware and software project that created an FPGA board programmed to behave like a ZX Spectrum computer. Over time the project grew and it’s now possible to use different software configurations (cores) that work like different systems other than the ZX Spectrum. You can choose your desired configuration from all those available.

This new project offers a more modern video output using DisplayPort, VGA and analogue audio (I2S or Sigma-Delta DAC) and is based on the Xilinx Artix7 board from QMTECH.
